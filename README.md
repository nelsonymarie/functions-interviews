# Reverse String Function

## Description
This Python function, `reverse_string`, efficiently reverses a given string using slicing. It takes a string as input and returns the reversed string. The function is implemented with Python's built-in slicing mechanism, making it concise and effective for reversing strings of any length.

## Usage
To use the `reverse_string` function, simply call it with a string argument. Here's an example:

```python
print(reverse_string("Hello World"))
